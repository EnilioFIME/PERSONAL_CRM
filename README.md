# Introduction

This project is personal.
A local CRM-like system for my dad, tailored to his needs and wants. He's a small real estate agent that wants to keep track on his prospects and appointments.
The purpose of this project is so he can streamline his workflow into one connected system available both in his phone and computer.

### As of right now...

He still uses a kind of "pen and paper" way to keep track of his appointments, basically he's got all of his notes organized into an excel spreadsheet that manually updates constatly - sometimes he would forget to update it more other than not, losing track on potential buyers.

# Scope

- A fullstack web application development, mobile first in mind, tailored to his personal needs and wants. 
- Small hosted and deployed in a service that will most probably stay free as its expected users is only him.
- Built on scalability in mind, as I want to sell this project as a SaaS in my locality.
